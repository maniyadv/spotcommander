UPGRADE NOTES:

 * Close SpotCommander completely on all your devices before upgrading
 * Android users: Make sure you are running v 3.5 or newer of the Android app
 * iOS users: If you are upgrading from v 8.8 or older, you should re-add to home screen
 * If you are upgrading from v 8.8 or older, make sure you reload/refresh the web app at least twice on every device

CHANGELOG:

 * Biggest release in a long time
 * Taking advantage of Spotify's new web API
 * Create playlists
 * Add tracks to playlists
 * Browse the starred playlist
 * Browse playlists with more than 200 tracks
 * Browse albums with more than 50 tracks
 * Library instead of Starred (currently only tracks can be managed through Spotify's web API)
 * Better browsing of artists
 * Artist biographies
 * Better search
 * Search results now include artists
 * Big improvements to Discover, with new Genres & moods
 * View your profile
 * Menu always visible if the screen is wide enough
 * Suspend and shut down the computer (in the now playing overflow menu)
 * Right-click on list items to show the actions overflow dialog
 * Many features now require authorization with Spotify
 * Removed custom font because of a bug in the Android 4.4 webview
 * Various UI changes
 * Bug fixes

KNOWN ISSUES:

 * Many of the playlists belonging to the user "spotify" can not be browsed. Bug verified by Spotify.
 * Playlists belonging to users with certain special characters in their username can not be browsed. Bug verified by Spotify.
